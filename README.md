# 여행 경비 관리 서비스 기능 명세서

## 1. 개요

여행 경비 관리 서비스는 사용자가 여행 중 발생하는 지출을 효율적으로 관리하고 분석할 수 있도록 도와주는 애플리케이션입니다. 이 서비스는 영수증을 통한 자동 입력, 수동 입력, 지출 내역 시각화, 평균 여행 지출 비교 등의 기능을 제공합니다.

## 2. 기능 설명

### 2.1. 영수증을 통한 지출 구분

### 기능 설명

- **영수증 스캔 및 업로드:** 사용자가 영수증을 사진으로 찍어 업로드할 수 있습니다.
- **자동 지출 항목 분류:** 업로드된 영수증의 내용을 OCR(광학 문자 인식) 기술을 통해 읽어내어 지출 항목을 자동으로 분류합니다.
- **카테고리 지정:** 식사, 교통, 숙박 등 다양한 카테고리로 자동 분류합니다.
- **수동 수정:** OCR 인식 결과가 정확하지 않은 경우 사용자가 수동으로 수정할 수 있습니다.

### 사용자 스토리

- 사용자 A는 여행 중 식사 후 영수증을 사진으로 찍어 애플리케이션에 업로드합니다. 애플리케이션은 영수증을 인식하여 식사 항목으로 자동 분류합니다.

### 2.2. 사용 현황과 평균 여행 지출 비용 비교

### 기능 설명

- **평균 지출 데이터베이스:** 다양한 여행 목적지와 여행 유형별 평균 지출 데이터를 제공합니다.
- **비교 기능:** 사용자의 현재 지출 내역을 동일한 조건의 평균 지출 내역과 비교하여 시각적으로 보여줍니다.
- **분석 보고서 제공:** 사용자의 지출 패턴을 분석하여, 절약할 수 있는 항목이나 과도한 지출 항목에 대한 인사이트를 제공합니다.

### 사용자 스토리

- 사용자 B는 유럽 여행 중 현재까지 사용한 비용이 평균 비용과 어떻게 다른지 알고 싶습니다. 애플리케이션에서 현재 지출 내역과 유럽 여행의 평균 지출 내역을 비교해봅니다.

### 2.3. UI를 통한 지출 내역 시각화

### 기능 설명

- **대시보드 제공:** 전체 지출 내역을 한눈에 볼 수 있는 대시보드를 제공합니다.
- **그래프 및 차트:** 카테고리별 지출 비율, 일자별 지출 내역 등을 그래프와 차트로 시각화합니다.
- **필터 및 정렬:** 날짜, 카테고리, 금액 등 다양한 조건으로 지출 내역을 필터링하고 정렬할 수 있습니다.

### 사용자 스토리

- 사용자 C는 여행 경비를 시각적으로 확인하기 위해 애플리케이션을 열고 대시보드를 확인합니다. 여기서 각 항목별 지출 비율과 일자별 지출 내역을 그래프로 확인합니다.

### 2.4. 영수증이 없는 지출 내역 수동 기입

### 기능 설명

- **수동 입력 기능:** 영수증이 없는 지출 내역을 사용자가 직접 입력할 수 있는 인터페이스를 제공합니다.
- **카테고리 선택:** 수동으로 입력한 지출 내역도 자동 입력과 동일하게 카테고리로 분류할 수 있습니다.
- **노트 추가:** 사용자가 필요시 지출 내역에 대한 추가 노트를 입력할 수 있습니다.

### 사용자 스토리

- 사용자 D는 길거리에서 소액의 현금을 사용했지만 영수증이 없습니다. 애플리케이션에서 해당 지출 내역을 수동으로 입력하고 "기념품 구매" 카테고리로 지정합니다.

## 3. 기술 사양

- **프론트엔드:** React.js
- **백엔드:** Node.js, Express.js
- **데이터베이스:** MongoDB
- **OCR 기술:** Tesseract.js
- **비교 분석:** 데이터베이스 내 평균 지출 데이터와 비교
- **UI/UX 디자인:** Figma, Adobe XD

## 4. 결론

이 여행 경비 관리 서비스는 여행자들이 지출을 체계적으로 관리하고 분석할 수 있도록 돕습니다. 자동화된 영수증 처리, 지출 내역 시각화, 평균 비용 비교 등의 기능을 통해 사용자의 편의성을 극대화하고 효율적인 지출 관리가 가능하도록 설계되었습니다.
